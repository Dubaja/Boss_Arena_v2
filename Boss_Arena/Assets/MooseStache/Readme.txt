The Pixel Top-Down Shooter Engine e is a complete Unity 2D top down shooter engine.
Basically it'll allow you to create your own platformer game.
It comes complete with lots of visual assets, animations, particle effects, etc...

WHAT AM I SUPPOSED TO DO WITH IT ?
----------------------------------

The PIXEL TOP-DOWN SHOOTER ENGINE includes everything you need to start your own 2d topdown shooter game.
You can either start fresh and pick the scripts you need from it, or modify the existing demo levels brick by brick, adding your own sprites and scripts as you go.
